Comandos de Terminal

¿Qué es una terminal?

La terminal es una interfaz de texto que permite interactuar con el sistema operativo. A través de ella se pueden ejecutar comandos para realizar tareas como mover, copiar, borrar archivos, instalar programas, etc.

Comandos de la terminal

Algunos comandos básicos de la terminal son los siguientes:

Comando Descripción
ls Muestra los archivos y directorios dentro del directorio actual.
cd <directorio> Cambia al directorio indicado
cd .. Regresa al directorio anterior
pwd Muestra la ruta del directorio actual.
mkdir <directorio> Crea un directorio.
rmdir <directorio> Elimina un directorio.
rm -rf <directorio> Remueve un directorio y sus contenidos.
touch <archivo> Crea un archivo vacío.
rm <archivo> Elimina un archivo o directorio.
cp <archivo> <directorio> Copia un archivo o directorio.
mv <archivo> <directorio> Mueve un archivo o directorio.
cat <archivo> Muestra el contenido de un archivo.
less <archivo> Muestra el contenido de un archivo, página por página.
head <archivo> Muestra las primeras líneas de un archivo.
tail <archivo> Muestra las últimas líneas de un archivo.
find <ruta> -name <patrón> Busca archivos o directorios que coincidan con un patrón.
locate <patrón> Busca archivos o directorios que coincidan con un patrón.
grep <patrón> <archivo> Busca un patrón en el contenido de un archivo.
man <comando> Muestra el manual de un comando.
<comando> --help Muestra la ayuda de un comando.
vim <archivo> Abre un archivo en el editor de texto Vim.
clear Limpia la pantalla de la terminal.
history Muestra el historial de comandos.
exit Cierra la terminal.
