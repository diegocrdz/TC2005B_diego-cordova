CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para dar cualidades estéticas a una página web. El código CSS se puede ligar a documentos con código HTML (utilizado para crear el contenido y estructura de una web) para darle estilo a la página con colores, fuentes, márgenes, rellenos, etc.

CSS es denomminado como estilos en cascada debido a que el código se lee y aplica de arriba hacia abajo.

¿Cómo se utiliza CSS?

CSS se utiliza bajo el conepto de separar la estructura y presentación de una página web, haciendo que el código sea más fácil de modificar y mantener a largo plazo. La idea es mantener la siguiente estructura:

  • Archivos HTML: Estructura y contenido de la página web.
  • Archivo CSS: Diseño de la página web (colores, fuentes, márgenes. etc.).

Idealmente, se busca utilizar un único archivo CSS para cada página web, modificando los aspectos visuales de toda la página en un solo lugar.

¿Cómo ligar un CSS a un HTML?

Para ligar un archivo CSS a un archivo HTML, se utiliza la etiqueta <link> en la sección <head> del archivo HTML. Se agrega el atributo href para indicar la ruta del archivo CSS. 

<link rel="stylesheet" href="archivo.css">

Selectores

Los selectores son la primera parte de una regla CSS. Son la forma en la que se indica a qué elementos de la página HTML se les aplicará un estilo.

Los selectores tienen un orden de precedencia; es decir, el orden en el que se aplican los estilos a los elementos de la página:

  1. Selectores de ID: 1-0-0
  2. Selectores de clase: 0-1-0
  3. Estilos de tipo: 0-0-1

Selectores de tipo

Los selectores de tipo se utilizan para seleccionar y aplicar estilos a todos los elementos de una etiqueta específica. Para esto, se utiliza el nombre normal de la etiqueta al definir el estilo:

En el ejemplo anterior, el selector h1 indica que se le aplicará el color azul y color de fondo amarillo a todos los elementos con la etiqueta <h1> del HTML al que esté conectado. Lo mismo ocurre con el color rojo para los elementos con etiqueta <p>.

Selectores de clase

Los selectores de clase se utilizan para seleccionar y aplicar estilos a un grupo de elementos con la misma clase. Para esto, se utiliza un punto antes del nombre de la clase al definir el estilo: 

.clase {
    color: blue;
}

Para aplicar la clase a un elemento HTML, se utiliza el atributo class: 

<p class="clase">Este párrafo pertenece a la clase "clase"</p>

Selectores de ID

Los selectores de ID se utilizan para seleccionar y aplicar estilos a un elemento único con un ID. Para esto, se utiliza el símbolo # antes del nombre del ID al que se aplicará un estilo: 

#id {
    color: red;
}

Para aplicar el ID a un elemento HTML, se utiliza el atributo id: 

<p id="id">Este párrafo pertenece al ID "id"</p>

Propiedades

Las propiedades son los valores que se le pueden asignar a los elementos de la página para modificar su apariencia. Algunas de las propiedades más comunes y que se utilizaron en este blog son las siguientes:

Propiedad Descripción Ejemplo
border Borde de un elemento border: 1px solid black;
border-left Borde izquierdo border-left: 1px solid black;
border-right Borde derecho border-right: 1px solid black;
border-top Borde superior border-top: 1px solid black;
border-bottom Borde inferior border-bottom: 1px solid black;
border-radius Radio de las esquinas border-radius: 10px;
background-image Imagen de fondo background-image: url("imagen.png");
color Color del texto color: red;
font Fuente del texto font: 16px sans-serif;
font-family Familia de la fuente font-family: sans-serif;
font-style Estilo de la fuente font-style: italic;
font-weight Grosor de la fuente font-weight: bold;
margin Margen del elemento margin: 10px;
padding Relleno del elemento padding: 10px;
overflow Manejo de desbordamiento overflow: hidden;

Valores

Los valores son una forma de definir subvalores admitidos; es decir, los los datos que se le asignan a las propiedades utilizadas para modificar la apariencia de los elementos de la página. Algunos ejemplos son los siguientes:

  • Colores: white, black o #ff0000
  • Fuentes: serif, sans-serif o monospace
  • Unidades: px (pixeles), em (ramaño de letra del elemento padre), vw (ancho de la ventana gráfica), cm (centímetros), etc.
